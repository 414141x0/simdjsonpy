# simdjsonpy

Typed Python 3.14+ bindings for [`simdjson`](https://simdjson.org) using [`nanobind`](https://github.com/wjakob/nanobind).

The detailed user and engineering guidance lives in the `mkdocs` site under [docs/](simdjsonpy/docs).


## Quick Commands

```bash
uv sync
uv run python -m unittest discover -s tests -v
mkdocs serve
```
